MiniFASNet 模型文件说明
======================

将预训练的 MiniFASNet 模型文件放在此目录下：

文件名: minifasnet.pth

如何获取模型：
1. 从公开的活体检测数据集训练
2. 使用预训练的 Silent-Face-Anti-Spoofing 模型
3. 或从相关 GitHub 项目下载

模型结构：
- 输入尺寸: 80x80x3 (RGB)
- 输出: 2类（真实人脸 / 伪造人脸）

注意：
- 如果没有预训练模型，系统会使用随机初始化的权重
- 随机初始化的模型检测效果不佳，仅供测试
- 建议使用训练好的模型以获得准确的活体检测结果
